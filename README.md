# Personal Portfolio üöÄ

This project is a modern portfolio built with **Next.js** and **Tailwind CSS** inspired by [Brittany Chiang](https://brittanychiang.com/).

## ‚ú® Features

- Home, Internship, Projects, Blog, and Contact sections
- Markdown-based blog with static generation
- Dark/Light theme toggle
- Smooth scroll-in animations
- Contact form powered by Nodemailer (or swap to Formspree)
- Visitor tracking to Google Sheets
- Ready for one-click deployment to **Vercel**

## üõ†Ô∏è Getting Started

1. **Install dependencies**

   ```bash
   npm install
   # or
   yarn
   ```

2. **Configure environment variables**

   Copy `.env.example` to `.env.local` and fill in your secrets.

3. **Run the development server**

   ```bash
   npm run dev
   ```

   Open `http://localhost:3000` to see the site.

4. **Add blog posts**

   Put markdown files in `content/posts/` with front-matter:

   ```md
   ---
   title: "Awesome Post"
   date: "2025-01-01"
   ---
   Content here‚Ä¶
   ```

5. **Deploy**

   Push to GitHub and click **Import Project** on Vercel. Add the same env vars in Vercel dashboard.

## üìß Contact Form

Uses Gmail SMTP via Nodemailer. If Gmail blocks less-secure apps, create an App Password or switch to Formspree.

## üìä Visitor Tracking

1. Create a Google Service Account with Sheets API enabled.
2. Share your Sheet with that service-account email.
3. Paste the sheet ID and service-account creds into `.env.local`.
